# Транспортная задача
Производственной компании необходимо разработать алгоритм оптимальной доставки товара. 

***
Исходные данные такие:

1. У компании 20 складов по всей России
2. Известна стоимость доставки груза между складами. Причем не всегда есть маршруты от одного до другого склада. Возможны ситуации, когда от склада A до склада B нужно доставлять через склад C, а прямого маршрута нет.

***
Необходимо разработать алгоритм, который на вход принимает 2 параметра: склад отправления и склад назначения и в ответ выдает минимальную стоимость доставки
Можно разработать на C#, C++ и пр. либо на псевдоязыке
